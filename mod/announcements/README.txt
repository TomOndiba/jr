The Elgg 1.x announcements plugin allows to add announcements for members and provide access to this via widget.


*Activating the announcements plugin*

Simply extract the announcements plugin into the mod directory and
activate it as usual using the Elgg 1.x tool administration.

This creates a "announcements" widget item in widgets list and menu item in user and administration area.

*Plugin settings*
Administrator user can use menu to open plugin setup page.
This page allow setup default number fo latest announcements to show in widget.

*Announcements administration*
Administrator can use "Manage announcements" menu item in administrative area to add new announcements or edit/delete exists.
